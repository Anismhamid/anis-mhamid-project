WM

סקירה כללית

פרויקט זה הוא מערכת מנהלי עובדים (WMS) שפותחה עבור לנהל ביעילות את נתוני העובדים, לעקוב אחר תשלומי השכר, ולהקל על ניהול רשומות באמצעות ממשק ידידותי למשתמש. הוא כולל חלון אחד, והוא נועד לייעל היבטים שונים של ניהול עובדים ביעילות.

Application Features
  • ניהול עובדים בקלות וביעילות
    • משתמשים יכולים להוסיף, להציג, לעדכן ולמחוק עובד
    • מעקב מדויק אחר נתוני השכר כולל ניכויים.
  • טכנולוגיות חדישות ומסד נתונים מאובטח
    • משתמש ב-MySQL ו- SQLAlchemy לאחסון וניהול נתונים יעילים
  • תמיכת API לאינטראקציה עם נתונים
    • מספק נקודות קצה RESTful API לגישה מאובטחת לנתונים ולמניפולציה.
  • גמישות בפריסה
    • ניתן לפרוס ב-Windows, macOS ו-Linux באמצעות PyInstaller או cx_Freeze.
    • בנוי עם Flask ו-Python לביצועים אופטימליים וקלות תחזוקה.
מבנה הקובץ
  • main_Screen קובץ Python הראשי המכיל את היגיון היישום
  • API מטפל בפעולות מסד נתונים
הוצאה לאור
Dependencies
  • Python 3.x
  • Tkinter, customtkinter, ttkbootstrap
  • Webbrowser, requests, pymysql, logging
  • Flask, flask_sqlalchemy, sqlalchemy
  • Datetime, openpyxl, docx, pptx, reportlab
