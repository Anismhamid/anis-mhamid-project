פרויקט זה הוא מערכת מנהלי עובדים (WM) שפותחה עבור ניהול ביעילות את נתוני העובדים, לעקוב אחר תשלומי השכר, ולהקל על ניהול רשומות באמצעות ממשק ידידותי למשתמש. הוא כולל חלון אחד, והוא נועד לייעל היבטים שונים של ניהול עובדים ביעילות.

  • ניהול עובדים בקלות וביעילות
  • משתמשים יכולים להוסיף, להציג, לעדכן ולמחוק עובד
  • מעקב מדויק אחר נתוני השכר כולל ניכויים
  • טכנולוגיות חדישות ומסד נתונים מאובטח
  • משתמש ב-MySQL ו-SQLAlchemy לאחסון וניהול נתונים יעילים
  • תמיכת API לאינטראקציה עם נתונים
  • מספק נקודות קצה RESTful API לגישה מאובטחת לנתונים ולמניפולציה
  • גמישות בפריסה
  • ניתן לפרוס ב-Windows, macOS ו-Linux באמצעות PyInstaller או cx_Freeze. בנוי עם Flask ו-Python לביצועים אופטימליים וקלות תחזוקה

--main_Screen-- קובץ Python הראשי המכיל את היגיון היישום
--API-- מטפל בפעולות מסד נתונים
Run the software

(1.) התקן LocalWP

How to Run the Software

  • Install LocalWP: Download and install LocalWP.
  • Create a Local Page: Create a new page in LocalWP.
  • Run the Local Site: Activate the page you created.
  • Open VS Code: Open the project folder in Visual Studio Code.
  • Configure Ports:
    • Open Api.py and replace the port with the port of your LocalWP page.
    • Repeat the process for main_screen.py.
  • Import Database: Open your Database and import the relevant file.
  • Run the Application:
    • Run Api.py and ensure it operates correctly.
    • Run main_screen.py to launch the application.
